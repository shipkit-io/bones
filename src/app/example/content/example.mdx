# Example Contentsxd

Welcome to the floating editor example! This content is editable when edit mode is enabled.

## Features

- **Markdown Support**: Write content using markdown syntax
- **Git Integration**: Changes are automatically committed
- **Pull Request Creation**: Significant changes create PRs
- **Edit Mode Toggle**: Enable/disable editing with a click

## Try it out!

1. Enable edit mode using the toggle button
2. Click anywhere in this content to start editing
3. Make some changes and click save
4. Check your Git history to see the commit
